# Capítulo 5: Repetições

### Exercício 5.1:
- Programa exibe os números de 1 a 100

### Exercício 5.2:
- Programa exibe os números de 50 a 100

### Exercício 5.3:
- Programa escreve a contagem regressiva do lançamento de um foguete

### Exercício 5.4:
- Programa retorna de 1 até o número digitado pelo usuário, mas, dessa vez, apenas os números ímpares.

### Exercício 5.5: 
- Reescrita do programa 5.4 para escrever os 10 primeiro múltiplos de 3.

### Exercício 5.6:
- Programa exibe os resultados no mesmo formato de uma tabuada: 2x1=2, 2x2=4,...

### Exercício 5.7:
- Programa pede que o usuário também digite o início e o fim da tabuada, em vez de começar com 1 e 10

### Exercício 5.8:
- Programa recebe dois números. Retorna o resultado da multiplicação do primeiro pelo segundo, usando apenas os operadores de soma e subtração

### Exercício 5.9:
- Programa recebe dois números. Retorna a divisão inteira do primeiro pelo segundo, assim como o resto da divisão. Utiliza-se apenas os operadores de soma e subtração para calcular o resultado.

### Exercício 5.11:
- Programa recebe o depósito inicial e a taxa de juros de uma poupança. Retorna os valores mês a mês para os 24 primeiros meses e o total ganho com juros no período.

### Exercício 5.12:
- Alteração do programa anterior de forma a perguntar também o valor depositado mensalmente. O valor será depositado no início de cada mês e é considerado para o cálculo de juros do mês seguinte.

### Exercício 5.14:
- Programa que recebe números inteiros do teclado. O programa lê os números até que o usuário digite 0. No final da execução, exibe a quantidade de números digitados, assim como a soma e a média artmética.

### Exercício 5.15:
- Programa controla uma pequena máquina registradora. Solicita ao usuário que digite o código do produto e a quantidade comprada. Utiliza-se a tabela de códigos para obter o preço de cada produto. Programa retorna o valor total das compras depois que o usuário digitar 0, qualquer outro código gera a mensagem de erro "Código inválido".
