# Programação Python

Repositório de estudo estruturado em Python, desenvolvido com base no livro *Introdução à Programação com Python* de Nilo Ney Coutinho Menezes. O conteúdo está organizado por capítulos e cobre desde os primeiros conceitos da linguagem até programação funcional e interface gráfica.

São mais de 50 atividades práticas distribuídas em seis capítulos, cada uma resolvida e commitada de forma incremental.

---

## Referência

**Livro:** Introdução à Programação com Python — Algoritmos e Lógica de Programação para Iniciantes, 2ª Edição  
**Autor:** Nilo Ney Coutinho Menezes

---

## Conteúdo por Capítulo

### Capítulo 2 - Preparando o Ambiente
Configuração do ambiente de desenvolvimento, instalação do Python e primeiros passos com o interpretador.

### Capítulo 3 - Variáveis e Entrada de Dados
Tipos de dados, declaração de variáveis, leitura de entrada do usuário e operações básicas.

### Capítulo 4 - Condições
Estruturas condicionais com `if`, `elif` e `else`. Operadores relacionais e lógicos aplicados a problemas práticos.

### Capítulo 5 - Repetições
Laços `while` e `for`, iterações sobre sequências, controle de fluxo com `break` e `continue`.

### Capítulo 6 - Listas
Criação e manipulação de listas, indexação, fatiamento, métodos nativos e aplicações com dados sequenciais.

## Como Usar

```bash
git clone https://github.com/cavalcantiraissas/Programacao-Python.git
cd Programacao-Python
```

Navegue até a pasta do capítulo desejado e execute os arquivos `.py` diretamente:

```bash
python nome-do-arquivo.py
```
